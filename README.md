# AI Test Page with locale URLs

This version is ready for GitHub Pages and supports these locale URLs:

- `/en/`
- `/no/`
- `/sv/`

## Structure

- `index.html` redirects to `/en/`
- `en/index.html`
- `no/index.html`
- `sv/index.html`
- `style.css`
- `script.js`

## GitHub Pages URL example

If your repository is named `repo`, the URLs will be:

- `https://YOUR-USERNAME.github.io/repo/en/`
- `https://YOUR-USERNAME.github.io/repo/no/`
- `https://YOUR-USERNAME.github.io/repo/sv/`

## Deploy

1. Push all files to the repository root.
2. Open **Settings → Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Select **main** and **/(root)**.
5. Save.
